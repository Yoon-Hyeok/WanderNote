# WanderNote

> **여행의 순간을 지도 위에 기록하는 감성 여행 다이어리 iOS 앱**

WanderNote는 단순한 사진 저장 앱을 넘어, 사용자가 여행 장소, 사진, 메모, 방문 날짜 등을 하나의 플랫폼에서 기록하고 지도 기반으로 시각화하여 관리할 수 있는 디지털 여행 다이어리입니다.

---

## 프로젝트 개요
현대인들은 여행을 하며 다양한 장소와 추억을 경험하지만, 시간이 지나면 세부적인 기억이 점차 흐려집니다. 대부분 사진만 저장하거나 SNS에 단편적으로 남기기 때문에 체계적인 관리가 어렵습니다. WanderNote는 이러한 문제를 해결하고 여행의 흐름과 감정을 직관적으로 되짚어 볼 수 있도록 돕습니다.

- **개발 환경**: Xcode, iOS 17.0+
- **주요 기술**: SwiftUI, SwiftData, MapKit, PhotosUI, Charts

---

## 주요 기능

### 1. 홈 (Home)
- **최근 여행 및 즐겨찾기**: 가장 최근에 다녀온 여행 기록을 가로 스크롤 카드로 확인합니다. 하트 아이콘을 눌러 즐겨찾기한 여행만 따로 모아볼 수 있습니다.
- **스마트 검색**: 장소명, 작성한 메모는 물론 '도시 이름'으로도 지난 여행을 쉽게 검색할 수 있습니다.
- **요약 통계**: 총 여행 횟수와 방문한 도시 수를 한눈에 파악할 수 있습니다.

### 2. 지도 시각화 (Map)
- **지도 위 여행 기록**: `MapKit`을 활용해 내가 방문했던 모든 장소를 지도 위에 핀(마커)으로 표시합니다.
- **상세 정보 확인**: 지도 위의 핀을 탭하면 하단에 여행 정보 카드가 올라오며, 클릭 시 상세 기록을 볼 수 있습니다.

### 3. 여행 기록 추가 및 수정 (Add / Edit)
- **스마트 사진 메타데이터 추출**: `PhotosUI`와 `Photos` 프레임워크를 연동하여, 갤러리에서 사진을 선택하면 **사진이 촬영된 날짜와 GPS 위치 정보(위도, 경도, 도시명)를 자동으로 불러와 입력**해 줍니다.
- **위치 직접 검색**: `MKLocalSearch`를 이용해 장소명이나 주소를 검색하여 방문한 위치를 정확하게 지정할 수 있습니다.
- **다양한 정보 기록**: 장소명, 방문 날짜, 여행 메모, 별점(1~5점)을 함께 저장하여 당시의 감정을 생생하게 기록합니다.

### 4. 프로필 및 여행 통계 (Profile)
- **다양한 통계 제공**: 내 여행 데이터를 바탕으로 다채로운 통계를 제공합니다.
- **월별 여행 차트**: `Charts` 프레임워크를 활용해 연도별/월별 여행 횟수를 막대 그래프로 한눈에 확인합니다.
- **도시별 여행 모아보기**: 내가 방문했던 도시 목록을 방문 횟수 순으로 정렬하여 보여주며, 특정 도시를 선택하면 해당 도시에서의 모든 기록을 모아볼 수 있습니다.

## 기술 스택

| Category | Frameworks |
| --- | --- |
| **UI** | `SwiftUI` |
| **Database** | `SwiftData` (`@Model`, `@Query`) |

| **Media** | `PhotosUI`, `Photos` (Asset Metadata) |
| **Data Visualization** | `Charts` |

---

## 화면 설계
- **홈 화면**: 최근 여행 표시, 통계 카드 표시, 검색 및 즐겨찾기 필터
- **지도 화면**: 지도 및 여행 위치 핀 표시, 핀 선택 시 정보 카드 팝업
- **추가 화면**: 갤러리 사진 선택 (날짜/위치 자동 기입), 장소명, 방문 날짜, 메모, 별점 작성
- **프로필 화면**: 여행 통계 요약, 연도별 월별 여행 차트, 방문한 도시별 기록 조회
